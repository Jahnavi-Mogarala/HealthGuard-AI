# HealthGuard AI

**AI-Driven Public Health Assistant for Disease Awareness & Community Safety**
*Tagline: Know Early. Stay Safe. Save Lives.*

HealthGuard AI is a production-quality full-stack web application designed for the Smart India Hackathon. It is a multilingual AI-powered Public Health Awareness Platform that provides verified disease awareness, outbreak intelligence, symptom education, emergency guidance, preventive healthcare, and more.

## Objective
To build a scalable, responsive, and accessible platform serving as an actual government-health portal mixed with AI-driven capabilities.

## Tech Stack

### Frontend
- Next.js 15
- React 19
- TypeScript
- Tailwind CSS & Shadcn UI
- Framer Motion

### Backend
- FastAPI (Python)
- PostgreSQL
- Redis
- LangChain & LangGraph
- Qdrant (Vector Database)

### DevOps & Architecture
- Docker & Docker Compose
- GitHub Actions (CI/CD)

## Disclaimer
This is a public health awareness platform. It is NOT a disease diagnosis system. AI-driven responses are for educational and awareness purposes only.
